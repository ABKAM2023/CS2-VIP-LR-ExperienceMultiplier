# EN
**[VIP(CSSharp)] [LR] ExperienceMultiplier** - grants VIP players an experience multiplier.

In vip.json
```json
  "ExperienceMultiplier": 1.5
```

Translation:
```json
  "ExperienceMultiplier": "Experience Multiplier"

```
